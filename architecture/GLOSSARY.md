# Enterprise Glossary

## Bounded Context
A logical boundary that owns its domain model.

## Domain Event
Immutable business event emitted by a bounded context.

## IELNex
Shared enterprise platform.

## KrishiOx
Agriculture product built on IELNex.

## Praxoma
Healthcare coordination product built on IELNex.

## Open Host Service
Stable integration interface exposed to other contexts.
